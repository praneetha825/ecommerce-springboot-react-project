# E-Commerce Application

A full-stack E-Commerce application built using Spring Boot, PostgreSQL, React, and Vite.

## Features

* Product Management (Add, Update, Delete)
* Product Search
* Shopping Cart
* Order Management
* Product Image Upload
* Responsive User Interface

## Tech Stack

### Backend

* Spring Boot
* Spring Data JPA
* Maven

### Frontend

* React
* Vite
* Axios
* Bootstrap
* React Router DOM

### Database

* PostgreSQL

## Project Structure

### Backend

* Controller Layer
* Service Layer
* Repository Layer
* DTO Layer
* Entity Layer

### Frontend

* Components
* Context API
* Routing
* Search Functionality
* Cart Management

## Installation

### Backend

```bash
cd "Get all Orders"
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```

### Frontend

```bash
cd t-ecom
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```
